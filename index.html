Summary: Latitude vs X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, data was first pulled from OpenWeather API to retrieve data for over 500 cities

Once data is gathered, Matplotlib was used to compare different weather aspects vs latitude. Factores we looked at include: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
